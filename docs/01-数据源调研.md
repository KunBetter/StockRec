## 一、数据源总览

### 1.1 生产级推荐组合

```
Baostock（历史K线+财务） + AKShare（实时行情+资金流向） + 新浪/腾讯API（实时价格）
```

| 数据源 | 类型 | 实时行情 | 历史K线 | 分钟K线 | 财务数据 | 注册 | 费用 | 稳定性 |

| **Baostock** | Python库 | ❌ | ✅ | ✅ | ✅ | 无需 | 免费 | ★★★★★ |

### 1.3 数据源分类

**第一梯队（主力数据源）**
- Baostock：历史K线（日/周/月/分钟）、财务数据、复权因子、宏观经济
- AKShare：实时行情、资金流向、行业板块、龙虎榜、新闻舆情

**第二梯队（补充/冗余）**
- 新浪API：实时价格快照，零成本备份
- 腾讯API：实时价格快照，更新频率3秒

**第三梯队（未来升级路径）**
- Tushare Pro：更稳定的历史数据、研报数据
- TickFlow：低延迟分钟K线、实时行情
- XTick：Tick级数据、竞价数据

## 二、主力数据源详解

### 2.1 Baostock（历史数据核心）

**安装**：`pip install baostock`

**核心API**：
```python
import baostock as bs

# 登录
lg = bs.login()

# 日K线（含复权）
rs = bs.query_history_k_data_plus("sh.600000",
    "date,code,open,high,low,close,preclose,volume,amount,adjustflag,turn,tradestatus,pctChg,isST",
    start_date='2020-01-01', end_date='2025-12-31',
    frequency="d", adjustflag="2")  # 2=前复权

# 分钟K线
rs = bs.query_history_k_data_plus("sh.600000",
    "date,time,code,open,high,low,close,volume,amount,adjustflag",
    start_date='2024-01-01', end_date='2024-01-31',
    frequency="5")  # 5/15/30/60分钟

# 财务数据（利润表）
rs = bs.query_profit_data(code="sh.600000", year=2025, quarter=1)

# 财务数据（资产负债表）
rs = bs.query_balance_data(code="sh.600000", year=2025, quarter=1)

# 行业分类
rs = bs.query_stock_industry()

# 登出
bs.logout()
```

**注意**：
- pandas 2.0+ 需要兼容性处理
- 数据收盘后更新，无盘中实时数据
- 需 `bs.login()` 后才能调用

### 2.2 AKShare（实时+另类数据）

**安装**：`pip install akshare`

**核心API**：
```python
import akshare as ak

# A股实时行情
df = ak.stock_zh_a_spot_em()  # 东方财富实时行情

# 历史日K线
df = ak.stock_zh_a_hist(symbol="000001", period="daily",
    start_date="20200101", end_date="20251231", adjust="qfq")

# 资金流向
df = ak.stock_individual_fund_flow(stock="000001", market="sz")

# 龙虎榜
df = ak.stock_sina_lhb_detail_daily(trade_date="20250522")

# 新闻舆情
df = ak.stock_news_em(symbol="000001")

# 行业板块
df = ak.stock_board_industry_name_em()
```

## 三、数据获取策略设计

### 3.1 数据分层获取

```
Layer 1 - 实时层（秒/分钟级）
├── 新浪API → 全市场股票实时价格快照
└── AKShare spot_em → 补充实时行情

Layer 2 - 历史层（T+1更新）
├── Baostock → 日K线全量历史 + 财务数据
└── AKShare hist → 补充验证 + 资金流向

Layer 3 - 基础层（静态/慢变）
├── Baostock → 行业分类、股票基本信息
└── 本地DB → 缓存的全量基础数据
```

### 3.2 容错与降级

```
Baostock不可用 → 降级到 AKShare hist
AKShare不可用 → 降级到 新浪/腾讯API
新浪API不可用 → 降级到 腾讯API
全部不可用 → 使用本地DB缓存的最新数据 + 告警
```

### 3.3 更新频率

| 数据类型 | 更新频率 | 数据源 |
|----------|----------|--------|
| 实时价格快照 | 每小时（整点） | 新浪/腾讯API |
| 日K线增量 | 每日收盘后（15:30） | Baostock |
| 财务数据 | 每季度/每报告期 | Baostock |
| 资金流向 | 每日收盘后 | AKShare |
| 新闻舆情 | 每小时 | AKShare |
| 股票基本信息 | 每周 | Baostock |
