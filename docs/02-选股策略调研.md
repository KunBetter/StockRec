## 一、策略分类体系

### 1.1 按方法论分类

```
选股策略体系
├── 多因子选股（Alpha因子挖掘）
│   ├── 价值因子（PE/PB/PS/股息率）
│   ├── 质量因子（ROE/ROA/毛利率/存货周转）
│   ├── 动量因子（价格动量/换手率/波动率）
│   ├── 成长因子（营收增速/利润增速/研发占比）
│   ├── 量价因子（高频微观结构/资金流向）
│   └── 另类因子（新闻舆情/专利/线上销量）
│
├── AI/深度学习选股
│   ├── 树模型（LightGBM/XGBoost → 因子→收益映射）
│   ├── 时序模型（GRU/LSTM/BiLSTM → 历史序列预测）
│   ├── 注意力模型（Transformer → 多维度聚合）
│   └── 排序学习（LambdaRank → 头部排序优化）
│
├── 轮动策略
│   ├── 行业轮动（行业强弱轮动）
│   ├── 风格轮动（大小盘/价值成长）
│   └── 热点轮动（主题→龙头股映射）
│
└── 事件驱动
    ├── 短期事件（盈利超预期/高股息预案/指数纳入）
    └── 长期事件（增持计划/股票回购/股权激励）
```

## 三、推荐策略组合方案

### 3.1 策略组合原则

1. **多策略低相关**：组合相关性<0.3的策略，降低整体回撤
2. **分层覆盖**：不同策略覆盖不同市场环境（趋势/震荡/反转）
3. **频率错配**：日频+周频+事件驱动混合，平滑换手
4. **可解释性**：主要仓位用因子模型，辅助仓位用深度学习

### 3.2 StockRec 生产级三层策略

```
Layer 1：稳健基础层（权重60%）
├── 多因子选股（20+因子等权/ICIR加权）
├── 沪深300/中证500成分股范围内
└── 目标：稳定超额3-5%，最大回撤<15%

Layer 2：增强Alpha层（权重30%）
├── LightGBM+GRU深度学习模型
├── 全A选股范围
└── 目标：年化超额15-25%，控制小市值暴露

Layer 3：事件驱动层（权重10%）
├── 盈利超预期+高股息预案+增持回购
├── 事件触发型交易，低换手
└── 目标：年化超额10-15%，与市场低相关
```

### 3.3 排序与加权方案

选股流程：
1. 每层独立生成候选股票池和得分
2. 层内按得分从高到低排序
3. 层间按权重(60/30/10)加权融合
4. 行业中性化 + 市值中性化
5. 移除ST/停牌/涨跌停无法交易股票
6. 最终输出 Top 10-30 推荐股票

### 3.4 基于 DeepSeek Pro 的 AI 分析层

利用 DeepSeek Pro 大模型进行：
1. **财务报告解读**：解析最新季报/年报的业绩变化、管理层讨论、风险提示
2. **新闻舆情分析**：对近期新闻进行情感打分和影响评估
3. **行业趋势判断**：分析行业政策、景气度变化
4. **综合推荐理由**：为每只Top股票生成买入理由和风险提示
5. **策略信号校验**：验证量化模型的选股结果是否与基本面/消息面一致

## 四、风险控制

| 维度 | 措施 |
|------|------|
| **行业集中度** | 单一行业占比 ≤ 25% |
| **个股权重** | 单只 ≤ 10%，Top3合计 ≤ 30% |
| **市值分布** | 大/中/小盘配比 50/30/20 |
| **止损机制** | 单只-15%止损，组合-10%减半仓 |
| **市场风险** | 大盘跌破200日均线 → 减仓至50%以下 |
| **因子失效** | 每月检验因子IC，连续2月IC<0 → 剔除 |
