# StockRec 初期变现方案（无需金融牌照）

## 一、背景与前提

StockRec 当前处于早期阶段，尚未取得金融牌照（投顾牌照、基金代销牌照等）。在此阶段，所有变现手段必须满足两个硬约束：

1. **不涉及证券投资咨询** — 不对个股给出买卖建议，不构成《证券投资咨询业务暂行规定》中定义的"投资建议"
2. **不触碰基金/理财代销** — 不直接销售或分销金融产品

在此前提下，依然有三条可行的变现路径。

---

## 二、路径一：接入广告联盟（信息流广告）

### 2.1 方案概述

在 App 的信息流（资讯流、自选列表、搜索结果页等）中嵌入原生广告，通过穿山甲（字节跳动）和/或腾讯广告联盟（优量汇）进行程序化变现。

### 2.2 接入平台对比

| 维度 | 穿山甲（Pangle） | 优量汇（腾讯广告） |
|------|-----------------|-------------------|
| 填充率 | 高（DAU 基数大） | 中高 |
| eCPM | ¥5-30（视品类而定） | ¥3-25 |
| SDK 体积 | ~2MB | ~3MB |
| 广告形式 | 原生、开屏、插屏、激励视频 | 原生、开屏、插屏、激励视频、Banner |
| 审核门槛 | 较低，App 上架即可 | 需有 ICP 备案 |
| 结算周期 | 月结（满 ¥100 起付） | 月结（满 ¥100 起付） |

### 2.3 建议广告位布局

| 广告位 | 形式 | 预估 eCPM | 用户干扰度 | 优先级 |
|--------|------|-----------|-----------|--------|
| 资讯信息流 | 原生信息流广告（图文/视频），穿插在第 4-6 条内容后 | ¥8-20 | 低 | 高 |
| 自选页底部 | 原生 Banner，不遮挡核心数据 | ¥3-8 | 极低 | 高 |
| 开屏广告 | 全屏 3-5 秒，可跳过 | ¥15-30 | 中 | 中 |
| 搜索结果页 | 信息流广告，穿插在搜索结果中 | ¥5-15 | 低 | 中 |
| 激励视频 | 用户主动观看以解锁增值功能（如查看更多指标） | ¥20-50 | 无（用户主动触发） | 低 |

### 2.4 收入估算

假设 DAU 5 万、人均信息流曝光 8 次/天、填充率 80%：

```
日曝光量 = 50,000 × 8 × 80% = 320,000 次
日收入   = 320,000 × ¥10（平均 eCPM）/ 1000 ≈ ¥3,200/天
月收入   ≈ ¥96,000
```

关键变量：DAU 规模、用户留存率、广告填充率。当 DAU 达到 10 万时，月收入可突破 ¥15-20 万。

### 2.5 实施要点

- **频控策略**：同一用户每小时最多展示 2-3 条广告，避免过度干扰导致留存下降
- **A/B 测试**：对比不同广告位、频次对留存率和变现效率的影响，找到最优平衡点
- **合规红线**：广告不得使用"推荐牛股""稳赚"等虚假宣传话术（《广告法》第 28 条）
- **灰度上线**：先在 5%-10% 用户中测试，观察留存数据后再全量放开

## 三、路径二：券商开户导流（CPA 结算）

### 3.1 方案概述

与券商合作，将用户导流至券商开户页面（H5 落地页或券商 App 下载），按有效开户数结算（CPA, Cost Per Action）。这是金融类 App 早期阶段变现效率最高的路径之一。

### 3.2 合作模式

```
用户（StockRec）→ 点击"开户"入口 → 跳转券商开户 H5/App
  → 用户完成开户 + 入金 → 券商回传转化数据 → StockRec 获得 CPA 佣金
```

### 3.3 主流合作券商及 CPA 参考价

| 券商 | CPA 单价（参考） | 开户门槛 | 备注 |
|------|-----------------|---------|------|
| 华泰证券（涨乐财富通） | ¥80-150 | 入金 ¥1000+ | 品牌好，转化率高 |
| 中信证券 | ¥100-200 | 入金 ¥5000+ | 单价高但门槛也高 |
| 国泰君安 | ¥80-150 | 入金 ¥1000+ | 渠道政策稳定 |
| 广发证券 | ¥60-120 | 入金 ¥1000+ | 中小券商中转化率高 |
| 华宝证券 | ¥50-100 | 入金 ¥500+ | 门槛低，适合小白用户 |
| 老虎证券 | ¥80-180 | 入金 $500+ | 主打港美股 |
| 富途证券 | ¥80-200 | 入金 $500+ | 港美股龙头，用户认知强 |

### 3.4 产品设计建议

**入口布局：**

- **行情/自选页**：在股票详情页底部放置"开通账户，即刻交易"入口（强关联场景，转化率最高）
- **个人中心**：固定"开户福利"banner
- **推送触达**：用户连续浏览某只股票 N 天 → 推送"XX 股票你关注很久了，开通账户即可交易 👉 享 ¥XXX 新手福利"

**转化漏斗（参考值）：**

```
曝光 10,000 次 → 点击 500 次（CTR 5%）
  → 进入开户流程 200 人（跳转率 40%）
  → 完成开户 100 人（完成率 50%）
  → 完成入金 40 人（入金率 40%）

有效转化数 = 40 户
CPA 收入 = 40 × ¥120（取中位） = ¥4,800
```

### 3.5 收入估算

假设 DAU 5 万，日均开户入口曝光 1 万次：

```
日均有效开户数 ≈ 40 户（按上述漏斗）
日 CPA 收入 ≈ ¥4,800
月 CPA 收入 ≈ ¥144,000
```

实际转化率取决于用户质量。股票类 App 用户交易意愿强，开户转化率通常高于泛理财类 App。关键指标：

- **CTR**（入口点击率）：3%-8%
- **开户完成率**：40%-60%
- **入金转化率**：30%-50%
- **CPA 结算周期**：通常月结，部分券商支持半月结

### 3.6 实施要点

- **合规声明**：入口处标注"开户服务由 XX 券商提供，StockRec 仅提供信息展示"，不构成开户推荐
- **避免诱导话术**：不使用"必涨""稳赚"等承诺性语言
- **数据对接**：需与券商完成 API/回调对接，确保转化数据准确回传；也可通过第三方效果追踪平台（如 AppsFlyer、Adjust）完成归因
- **多家接入**：同屏展示 2-3 家券商供用户对比选择（费率、佣金、新手福利），既提升用户体验，也形成竞价机制推高 CPA 单价
- **A/B 测试**：对比不同入口文案和位置的转化率

## 四、路径三：非金融品牌广告

### 4.1 方案概述

面向非金融行业品牌主销售广告资源，通过品牌曝光（CPM）或效果投放（CPC/CPA）变现。股票类 App 用户的典型画像（25-45 岁、中高收入、男性偏多）对多个品类的品牌主具有吸引力。

### 4.2 目标广告主行业

| 行业 | 匹配逻辑 | 广告形式建议 |
|------|---------|-------------|
| 汽车（新能源为主） | 中高收入男性用户，购车决策者 | 开屏、信息流大图、品牌专区 |
| 3C/数码 | 科技敏感型用户，换机频率高 | 信息流图文、测评类软文 |
| 高端白酒/洋酒 | 商务社交场景匹配 | 开屏、品牌冠名 |
| 手表/奢侈品 | 高净值用户匹配 | 开屏、品牌专区 |
| 商旅/航空/酒店 | 商务出行高频用户 | 原生信息流 |
| 职业教育/MBA | 职场上升期用户 | 信息流、激励视频 |
| 男士护肤/男士服饰 | 男性用户占比高 | 信息流图文 |
| 保险（非理财型） | 家庭责任期用户（重疾险、医疗险） | 原生信息流 |

### 4.3 售卖方式

| 方式 | 结算 | 适用阶段 | 说明 |
|------|------|---------|------|
| 直客销售 | CPM/CPT（按时段买断） | DAU > 10 万后 | 需自建商务团队，毛利高但门槛高 |
| 代理销售 | 分成（通常代理拿 30%-50%） | 任何阶段 | 对接 4A 或本地广告代理商，省去商务团队成本 |
| 程序化（PDB/PD） | CPM | 早期可用 | 通过广告联盟的 PDB（Programmatic Direct Buy）模式让品牌主定向投放 |

### 4.4 建议路线

**早期（DAU < 10 万）**：
- 不设自有商务团队
- 对接 1-2 家广告代理商，由代理商打包售卖
- 同时通过穿山甲/优量汇的 PDB 模式承接品牌预算

**中期（DAU 10-50 万）**：
- 建立 1-2 人商务团队
- 直接对接汽车、3C 头部品牌
- 开屏 CPM 定价 ¥30-60（行业参考）
- 品牌专区 ¥5-15 万/月（行业参考）

### 4.5 收入估算

**早期（代理模式，DAU 5 万）**：

```
月广告位填充率 ≈ 30%（品牌广告非标，填充率低于程序化）
月品牌广告收入 ≈ ¥3-8 万（视代理商资源和议价能力）
```

**中期（直客 + 代理，DAU 20 万）**：

```
开屏 CPT（按天买断）：¥3,000-8,000/天 × 15 天/月出售 = ¥4.5-12 万
品牌专区：2-3 家 × ¥5 万/月 = ¥10-15 万
代理售卖：¥5-10 万/月
月品牌广告收入 ≈ ¥20-40 万
```

## 五、三路径对比与优先级

| 维度 | 信息流广告（联盟） | 券商开户导流 | 非金融品牌广告 |
|------|------------------|-------------|---------------|
| 启动门槛 | 低（接入 SDK 即可） | 中（需商务对接 + 数据回传） | 中高（需商务资源） |
| 收入天花板 | 中（取决于 DAU × eCPM） | 高（CPA 单价高，用户精准） | 高（品牌溢价空间大） |
| 用户体验影响 | 低-中 | 低（场景匹配，用户需求真实） | 低（品牌广告体验好于效果广告） |
| 合规风险 | 低 | 中（需注意话术，不构成开户建议） | 极低 |
| 人力投入 | 低（接入即自动运行） | 中（需维护券商合作关系） | 中-高（代理商管理或直客拓展） |
| 收入稳定性 | 中（受 eCPM 季节性波动） | 中（受行情影响，牛熊差异大） | 中低（非标广告，填充率波动） |

### 建议优先级

**第一优先：信息流广告 + 券商开户导流同时启动（Month 1-2）**

- 信息流广告接入最快（1 周可完成 SDK 接入），先确保有持续收入流
- 同步推进 2-3 家券商合作（商务谈判 + 技术对接约 3-4 周）

**第二优先：非金融品牌广告（Month 3-6）**

- 当 DAU 积累到一定规模且券商导流稳定后，启动品牌广告方向
- 先通过代理模式切入，降低前期商务投入

## 六、整体收入预估

假设以 DAU 5 万为基准，三个路径全部运转后：

| 收入来源 | 保守估计（月） | 乐观估计（月） |
|---------|--------------|--------------|
| 信息流广告（联盟） | ¥6 万 | ¥12 万 |
| 券商开户导流（CPA） | ¥8 万 | ¥18 万 |
| 非金融品牌广告 | ¥3 万 | ¥8 万 |
| **合计** | **¥17 万** | **¥38 万** |

DAU 增长至 10 万后，月收入有望达到 ¥30-60 万。

关键增长杠杆：
1. **DAU 是唯一最重要的变量** — 所有变现效率都与用户规模线性正相关
2. **券商导流转化率** — 优化开户入口位置和文案，提升漏斗每一环的转化
3. **广告填充率** — 联盟填充率 + 品牌广告填充率共同决定收入上限

## 七、风险与注意事项

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 广告过多损害用户体验与留存 | 长期 DAU 下降 | 严格频控 + A/B 测试找最优广告密度 |
| 券商合作中断或 CPA 降价 | 收入波动 | 同时接入 3 家以上券商分散风险 |
| 熊市用户流失、交易意愿下降 | 开户转化率大幅下滑 | 熊市侧重信息流广告和品牌广告收入 |
| 合规/监管收紧 | 业务被迫调整 | 所有文案由法务审核，不碰投资建议红线 |
| 广告联盟账号被封 | 收入中断 | 同时接入穿山甲和优量汇，互为备份 |

## 八、下一步行动

1. **技术侧**：评估穿山甲/优量汇 SDK 集成方案，预估开发工时
2. **商务侧**：梳理目标券商清单，启动首轮商务接触
3. **产品侧**：设计广告位原型，定义频控策略参数
4. **合规侧**：委托律师审核广告文案和开户导流话术的合规边界
5. **数据侧**：搭建广告收入 dashboard 和券商转化漏斗追踪体系
